2. Variables: 
   Names we give to certain values in our program.
   Containers for data.
   Assignment - process of storing a value inside a variable.
   Expression - a combination of numbers, symbols or other variables that produce a result when evaluated.

   Variables Naming Restrictions:
    > Don't use keywords or functions that Python reserves for its own.
    > Don't use spaces 
    > Must start with a letter or an underscore.
    > Must be made up of only letters, numbers and underscores.

3. Implicit Conversion:
  Interpreter automatically converts one data type into another.
